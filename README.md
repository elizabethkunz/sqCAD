# Superconducting Quantum Circuit Analysis & Design (sqCAD)  ![Version](https://img.shields.io/github/v/release/elizabethkunz/sqcad) ![License](https://img.shields.io/github/license/elizabethkunz/sqcad)

sqCAD: Superconducting Quantum Circuit Analysis &amp; Design is an open-source project aimed at providing researchers with resources to do circuit-level simulation of quantum devices.


## Installation

You can install sqcad using pip:

pip install sqcad
